# Portfolio

Personal developer portfolio — static site (HTML, CSS, vanilla JS).

## Local preview

```bash
python3 -m http.server 8765
```

Open [http://localhost:8765](http://localhost:8765).

## GitHub Pages

After pushing, enable **Settings → Pages → Branch: main, folder: / (root)**. The site will be available at `https://<username>.github.io/<repo>/`.

Add `resume.pdf` in this folder if you use the Download Resume button.
